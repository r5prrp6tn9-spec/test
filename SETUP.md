# 🎮 Chibi Bot - Бот-коллекционер чибиков для Telegram

## Что это?

Это Telegram-бот для коллекционирования чибиков из разных вселенных (Звездные войны, Во все тяжкие, Суперы). Ты можешь:

✨ **Получать** новых чибиков каждые 4ч 45м  
🪎 **Отправлять** их в рейды на локации за добычу  
🛠️ **Крафтить** предметы для улучшения статов  
📦 **Собирать** ресурсы и экипировать чибиков  
💰 **Зарабатывать** коины в боях

## Как запустить?

### 1️⃣ Подготовка

Тебе понадобится:
- **Python 3.11+**
- **MongoDB Atlas** аккаунт (бесплатно)
- **Telegram Bot Token** от @BotFather
- **Render.com** аккаунт (для хостинга)

### 2️⃣ MongoDB Setup

1. Зайди на [MongoDB Atlas](https://www.mongodb.com/cloud/atlas)
2. Создай новый кластер (M0 Free tier)
3. Добавь IP адрес твоей машины в Network Access
4. Скопируй CONNECTION STRING вида:
   ```
   mongodb+srv://username:password@cluster.mongodb.net/chibibot?retryWrites=true&w=majority
   ```

### 3️⃣ Bot Token

1. Напиши @BotFather в Telegram
2. Напиши `/newbot` и следуй инструкциям
3. Скопируй полученный токен

### 4️⃣ Локальная установка

```bash
# Клонируй репозиторий
git clone <твой-репо>
cd Chibi-bot-main

# Установи зависимости
pip install -r requirements.txt

# Создай .env файл
cp .env.example .env

# Отредактируй .env и добавь:
BOT_TOKEN=твой_токен_сюда
MONGODB_URI=твоя_строка_подключения_сюда
```

### 5️⃣ Локальный тест

```bash
python bot.py
```

Если видишь "ChibiBot initialized successfully" - все работает!

### 6️⃣ Деплой на Render

1. Залей код на GitHub
2. Зайди на [Render.com](https://render.com)
3. Создай новый Web Service
4. Выбери GitHub репозиторий
5. Добавь environment variables:
   - `BOT_TOKEN`
   - `MONGODB_URI`
6. Deploy!

## Система рейдов

### Как работает?

1. Выбираешь локацию (🟢 Легко, 🟡 Средне, 🔴 Трудно)
2. Выбираешь чибика из инвентаря
3. Он уходит в рейд на время:
   - **Легко**: 8-13 минут
   - **Средне**: 10-15 минут
   - **Трудно**: 15-21 минуту

4. После возврата получаешь **лут** и **опыт**

## Проблемы и решения

### Бот не запускается
```
❌ MONGODB_URI not found
✅ Проверь .env файл и наличие BOT_TOKEN
```

### Рейды не работают
```
❌ Chibi not found
✅ Сначала получи чибика командой /chibi
```

### Монета MongoDB истекла
```
✅ Перейди в Atlas, обнови платежные данные
✅ Или создай новый бесплатный кластер
```

## Зачем нужна MongoDB?

Все данные игроков (чибики, коины, рейды) хранятся в облачной БД MongoDB. Это позволяет:
- ✅ Сохранять прогресс между запусками
- ✅ Поддерживать тысячи игроков одновременно
- ✅ Не терять данные при перезагрузке
- ✅ Масштабировать без проблем

## Развертывание на Render (бесплатно)

Render дает тебе:
- ✅ Бесплатный хостинг
- ✅ Автоматический деплой из GitHub
- ✅ Мониторинг через UptimeRobot
- ⚠️ Гашение через 15 минут без трафика (upgrade при необходимости)

### Шаги:

1. Залей код на GitHub (приватный или публичный)
2. На Render.com создай **New → Web Service**
3. Подключи GitHub репозиторий
4. Установи **Build Command**: `pip install -r requirements.txt`
5. Установи **Start Command**: `python bot.py`
6. Добавь Environment Variables
7. Deploy!

## Мониторинг с UptimeRobot (бесплатно)

Чтобы бот не заснул на Render:

1. Зайди на [uptimerobot.com](https://uptimerobot.com)
2. Создай новый HTTP(s) Ping check
3. Вставь URL твоего Render сервиса
4. Интервал: каждые 5 минут
5. Сохрани

Готово! Бот не будет спать 🤖

## Поддержка

Если что-то сломалось:

1. Проверь логи: `python bot.py` (увидишь ошибки)
2. Проверь MongoDB подключение в Atlas
3. Убедись что токен бота правильный
4. Перезагрузи бот: `kill` процесс и заново `python bot.py`
