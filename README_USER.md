## 📊 Полная функциональность

### Команды (8 основных)

| Команда | Результат |
|---------|-----------|
| `/start` | Регистрация новичка |
| `/chibi` | Случайный чибик каждые 4ч 45м |
| `/collections` | Выбрать коллекции для дропа |
| `/cantina` | Открыть 24 локации для рейдов |
| `/craft` | Крафтить 15 предметов улучшения |
| `/warehouse` | Склад, инвентарь, ресурсы |
| `/bonus` | Ежедневный бонус (50-150 💰) |
| `/wipe` | Админ: полный вайп (с паролем) |

### Система рейдов

- 24 локации (легко/средне/трудно)
- Время рейда: 8-21 минут в зависимости от сложности
- Случайный лут от 1-3 предметов
- Система травм чибиков (требует лечения)

### Крафтинг

- 15 предметов с рецептами
- Улучшают статы: ❤️ здоровье, ⚡️ выносливость, 🗡️ урон
- Цена: 140-520 коинов в зависимости от предмета

### Экономика

- Ежедневный доход: 50-150 коинов
- Максимум ресурсов: 350/200/25 по категориям
- Полная система расходов и доходов

## 🛠️ Как использовать код

### Менять сообщения

Всё в `texts.py` - простой Python словарь:

```python
MESSAGES['welcome'] = "Новый текст приветствия"
BUTTONS['my_button'] = "📌 Новая кнопка"
```

### Добавить команду

1. В `texts.py` добавь текст
2. В нужный handler добавь метод
3. В `bot.py` зарегистрируй обработчик

### Масштабировать

- Добавляй локации в `LOCATIONS`
- Добавляй предметы в `CRAFTING_ITEMS`
- Всё просто: список → обработка → сохранение в БД
