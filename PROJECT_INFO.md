# 🤖 Chibi Bot - Проект переструктурирован

## 📋 Что включено в архив

### 🎮 Основной код (10 модулей)

| Файл | Размер | Что делает |
|------|--------|-----------|
| **bot.py** | 8.7 KB | Главный файл - регистрирует все команды и обработчики |
| **database.py** | 4.2 KB | Работа с MongoDB - CRUD операции для всех данных |
| **texts.py** | 17 KB | ВСЕ сообщения, кнопки, локации, предметы, конфиги |
| **utils.py** | 5.2 KB | Вспомогательные функции (формат времени, статы, лут и т.д.) |
| **collections_handler.py** | 2.9 KB | Система выбора коллекций для дропа чибиков |
| **raids.py** | 7.2 KB | Полная система рейдов с локациями и боями |
| **crafting.py** | 4.3 KB | Крафтинг 15 разных предметов с рецептами |
| **warehouse.py** | 6.5 KB | Управление инвентарем, экипировкой, ресурсами |
| **admin.py** | 4.0 KB | Админ команды (вайп, бан, разбан) |
| **daily.py** | 4.0 KB | Ежедневные чибики и бонусы |

**Всего кода: ~64 KB** (чистый, без комментариев)

## 🚀 Быстрый старт

### Вариант 1: Локально (в консоли)

```bash
# 1. Распакуй архив
unzip Chibi-bot-restructured.zip

# 2. Установи зависимости
pip install -r requirements.txt

# 3. Создай .env с твоими ключами
cp .env.example .env
# Отредактируй: BOT_TOKEN и MONGODB_URI

# 4. Запусти
python bot.py
```

### Вариант 2: На Render (облако)

```bash
# 1. Залей код на GitHub
git init && git add . && git commit -m "init" && git push origin main

# 2. На Render.com:
#    - New Web Service
#    - Выбери GitHub репо
#    - Раздел Environment добавь BOT_TOKEN и MONGODB_URI
#    - Deploy!

# 3. Отслеживай статус на UptimeRobot (монитор 5 мин)
```

## 🛡️ Система банов и прав

### Ролли в системе

- **Обычный игрок**: Все команды игрока
- **Админ** (@ya_admin7): Команды + админ функции
  - Бесконечные коины и чибики
  - Доступ к `/wipe` (с паролем)
  - Данные админа не сохраняются (всегда "заново")

### Баны

- **Причины**: Багоюзинг, спам, неадекватное поведение
- **Длительность**: 7 дней (разбанить можно вручную)
- **Проверка**: При каждой команде
